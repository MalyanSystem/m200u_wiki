# Print From Network Connection By Orcaslicer Software
 To connect to the network, you need to obtain the 
machine's IP address.





## Step 1: Connect Printer to Network

### On the Printer Display

1. First, on the machine's display screen, click the                              button.
<img id="print4-image" src="/img/print4.png" />

2. In the WLAN functionality, connect to the local network.
<img id="print5-image" src="/img/print5.png" />

3. Then click the button on the home screen.   
<img id="print6-image" src="/img/print6.png" />

4. Obtain the machine's IP address (as shown in the accompanying image)
<img id="print7-image" src="/img/print7.png" />



## Step 2: Connect Software to Printer

### In Your Slicing Software

1. After that, click the button in image ① on the slicing software. 
<img id="print8-image" src="/img/print8.png" />

2. In the text box of ②, enter the printer's IP address and click OK.  
<img id="print9-image" src="/img/print9.png" />


3. Click on ③ to check the connection status
<img id="print10-image" src="/img/print10.png" />




## Step 3: Print from Network

### Prepare Your Model

1. First, click the button for **File Addition (①) to import STL
 or 3MF format files. 
<img id="print1-image" src="/img/print1.png" />


2.  After successful import, click the button for**Slicing** (②) to complete the slicing process.
<img id="print2-image" src="/img/print2.png" />


3. After slicing is complete, click the button (Print), then click the button (Upload and Print) to transfer the G-code file to the printer.
<img id="print13-image" src="/img/print13.png" />



### Features of Network Connection Interface
<img id="print11-image" src="/img/print11.png" />

 1. Print History and Status: Allows you to reprint historical files and 
view print status.
 2. High-Definition Camera: Monitors the printed object.
 3. Motion Control: Allows adjustment of the extruder head along 
the X, Y and Z axes.
 4. Temperature: Displays the temperature of the extruder head and 
the  heated bed.
 5. Console: Allows input of printer control commands.
 6. Task List: Allows queuing, pausing, resuming, or canceling print 
tasks.

<img id="print12-image" src="/img/print12.png" />

 7. Fan Control: Allows adjustment of fan speed.
 8. Printer Limitations: Controls print speed.
