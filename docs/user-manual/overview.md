## Quick Reference

### Safety Guidelines

⚠️ **IMPORTANT SAFETY INFORMATION**

- Always operate in a well-ventilated area
- Keep flammable materials away from the printer
- Never touch the hotend or heated bed while printing
- Unplug printer before maintenance
- Use only recommended power supply
- Supervise first prints closely

### Printer Specifications Quick Reference

| Parameter | Value |
|-----------|-------|
| Build Volume | 180 × 180 × 180 mm |
| Device Size | 310 × 370 × 370 mm |
| Nozzle Size | 0.4 mm |
