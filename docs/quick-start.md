# Quick Start Guide

Welcome to your new Malyan M200 Ultra! This guide will help you safely unpack and verify all components.

## Unboxing

### Main Components
- ✅ 1x 3D printer (pre-assembled)
- ✅ 1x power cord
- ✅ 1x Instruction manual

### Accessories & Tools
- ✅ 3x Screw
- ✅ 1x Nozzle
- ✅ 1x Lubricant
- ✅ 4x Hex wrench
- ✅ 1x Teflon tube
- ✅ 1x Filament holder
- ✅ 1x Nozzle cleaning needle
- ✅ 1x Silicone protective case
- ✅ 1x Filament (Random Color) Print more with a spool of filament included.
- ✅ 1x Scraper blade (Use the 3D printer's built-in file to print the blade holder for installing the blade.)
<img id="Tools-image" src="/img/Tools.jpg" />
### Included Software

- ✅ Sample G-code files for testing

## Unboxing Steps

### Step 1: Open the Box Carefully
1. Cut the tape on all sides of the box
2. Remove the outer packaging
3. Check for any visible damage to the box

### Step 2: Remove Protective Materials
1. Carefully remove foam padding
2. Remove plastic wrap and protective covers
3. Save all packaging materials in case of returns

### Step 3: Inspect the Printer
1. Check for any visible damage to the printer frame
2. Verify that all belts are properly tensioned
3. Ensure the print bed is securely attached
4. Check that the extruder moves smoothly on rails

### Step 4: Verify All Components
Use the checklist above to ensure you have all items. Contact support if anything is missing or damaged.

## Initial Inspection Checklist

Before first use, check:

- [ ] Printer frame is intact with no cracks or bends
- [ ] All wires are properly connected and not damaged
- [ ] X, Y, Z axes move smoothly (power off, manual movement)
- [ ] LCD screen is not cracked
- [ ] Build plate is flat and not warped
- [ ] Nozzle is clean and clear
- [ ] All screws and bolts are tight

## Storage Recommendations

If you're not setting up immediately:
- Store in a dry, temperature-controlled environment
- Keep original packaging if possible
- Avoid exposing to direct sunlight
- Protect from dust and moisture

## Next Steps

After unboxing, proceed to:
1. [Quick Start Guide](/docs/getting-started/quick-start) - Get printing quickly
2. [Initial Setup](/docs/getting-started/initial-setup) - Complete setup instructions

# Quick Start Guide

Get your Malyan M200 Ultra up and running in just a few minutes. This quick start guide will help you print your first model.

## Prerequisites

Before you begin, make sure you have:
- ✅ Unpacked your printer and checked all components
- ✅ Found a stable, level surface for your printer
- ✅ Prepared a power outlet nearby
- ✅ Connect to Wi-Fi, enter the device’s IP address in a web browser to access the web control panel for file upload and printing, or use the built-in USB port to print files from your own USB drive.
- ✅ Download and install OrcaSlicer slicing software
- ⭐ Before starting the print, use a 2.5mm wrench to first remove the four screws on the fixing lock ([as shown in the figure](#lock2-image)).

## Step 1: Initial Setup

1. **Place the printer** on a stable, level surface
2. **Connect the power supply** to the printer and wall outlet
3. **Power on** the printer using the power switch
4. **Wait for initialization** - the OLED touchscreen will display startup information

## Step 2: Calibration

The Malyan M200 Ultra features **Automatic Bed Leveling** and **Auto Resonance Compensation** - no manual adjustment needed!

1. On the OLED touchscreen, Please follow the on-screen instructions to complete the "Calibration Wizard" after the first boot.
2. The printer will automatically level the build plate and measuring resonances.
3. Wait for the process to complete

## Step 3: Load Filament

1. On the OLED touchscreen, navigate to **Control**
2. Select **Extruder Head Temperature Setting** and set to 200°C (for PLA)
3. Wait for preheating to complete
4. Select **Load** icon
5. Follow the on-screen prompts to load filament
6. Watch for steady filament flow from the nozzle
<img id="load-image" src="/img/load.jpg" />
## Step 4: Your First Print

### Option A: Print from USB Storage

1. Open slicing software and configure for Malyan M200 Ultra (see [Software Installation](/docs/software/slicer-setup))
2. Load a simple test model 
3. Slice the model and export G-code to USB storage
4. Insert USB storage into printer
5. On printer touchscreen, navigate to **File Management**
6. Select your G-code file and start printing

### Option B: Print from Network (WLAN)

1. Connect printer to Wi-Fi network (see [Network Printing Guide](/docs/usage/network-printing))
2. Note the IP address displayed on home screen
3. Connect software to printer via network or enter the IP address in a web browser
4. Send print job directly from software to printer
5. Watch your first print come to life!
